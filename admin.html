How to Upload
STEP BY STEP GUIDE
```
1
Download the app file
Click Download Koino App. A file called koino_app.html will save to your Downloads folder. This file has ALL your questions baked in.
2
Open GitHub (or your host)
Go to github.com and open your Koino repository. If you don't have one yet, create a free account and a new repository called koino.
3
Upload the new file
In your repo, click Add file → Upload files. Drag in the koino_app.html file. If a file with that name already exists, GitHub will ask you to replace it — say yes.
4
Commit the change
Scroll down and click Commit changes (green button). Your app is now live within seconds.
5
Check your live link
Visit your GitHub Pages link — usually yourusername.github.io/koino/koino_app.html — and your new questions will be there. Hard refresh if you don't see them (Cmd+Shift+R on Mac, Ctrl+Shift+R on Windows).
```
